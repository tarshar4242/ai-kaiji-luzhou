這是一個專為**公部門零基礎同仁**設計的 AI 入門課程導覽頁，由 **🍀 Learn AI with Tarshar** 的「小D教練」製作。授課或分享時，只要點左側目錄，就能直接帶到對應的講解內容、示範與可複製的提示詞——讓教學不卡頓、學員跟得上。

## 這個頁面想解決什麼

零基礎的學員最怕兩件事：**畫面跳來跳去跟不上**、**回家就忘了怎麼用**。

所以這個導覽台把一整堂課收進**一頁**：
- 講師點目錄，講到哪、頁面就停在哪，不用切換好幾個檔案。
- 每個示範都附**可一鍵複製的提示詞**，學員當場貼上就能跑出結果。
- 課後這一頁就是「帶得走的工具箱」，學員回辦公室照著做即可。

## 課程主軸

以「**公務現場真的用得到**」為設計原則，帶四個貼近監理所日常的 AI 示範：

| 示範 | 解決的工作場景 |
|---|---|
| 🧊 破冰小幫手 | 跟 AI 第一次對話，破除「我不會」的心理門檻 |
| 📄 公文回覆 | 把民眾留言 / 陳情，整理成有同理心又得體的回覆 |
| 🗣️ 櫃台口譯 | 面對外籍民眾，即時中英互譯不慌張 |
| 📝 會議記錄 | 把零散討論整理成正式會議記錄與分工 |

搭配 **NotebookLM** 做資料閱讀與彙整，讓同仁回辦公室也能延續使用。

## 頁面特色

- **左側目錄**：分組、可點選、捲動自動高亮，授課時點哪講哪。
- **搜尋**：關鍵字即時篩選段落，臨場找內容很快。
- **可複製提示詞**：深綠提示框配一鍵複製鈕，學員當場就能用。
- **示範連結位**：每個示範可掛上「跑好的範例連結」，現場直接展示。
- **講解重點卡**：每段附上講師要強調的關鍵句，不漏重點。
- **手機友善**：目錄可收合，手機分享也好讀。

## 怎麼維護

這頁是**單一自包含的 `index.html`**，所有課程內容都存在檔案裡的 `sections` 資料陣列，改內容不用懂程式：

> 由講師提供內容 → 更新 `sections` → 自動佈署上線，網址不變。
